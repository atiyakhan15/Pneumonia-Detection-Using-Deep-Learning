# 🩺 Pneumonia Detection Using Deep Learning

## 📌 Overview
This project implements a transfer learning-based deep learning model for automatic pneumonia detection from chest X-ray images.

## 🔧 Tech Stack
- Python
- PyTorch
- DenseNet121
- MedMNIST
- Resnet50

## 📊 Results
- Accuracy: 91%
- ROC-AUC: 0.94
